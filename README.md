# PiDmxDevice - Français
## Dispositif esclave DMX512 basé sur Raspberry Pi
Ce projet propose un ensemble matériel/logiciel pour réaliser rapidement des éléments de scène pilotés par une console DMX512DMX512](https://fr.wikipedia.org/wiki/DMX_(%C3%A9clairage))
- éclairage LED personalisé
  - Strip LED RGB
  - LEDs disposées pour un rendu particulier, par exemple 7 LEDs pour la constellation de la Grande Ourse
- relais électro-mécanique
- asservissement motorisé

Le faible coût des boards Raspberry et la facilité de programmer, en Python par exemple, permettent de réaliser rapidement des éléments de scènes particuliers.
